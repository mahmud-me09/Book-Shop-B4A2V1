### How to setup the project locally:

1. Clone the repository.
2. create a environment variable file with the following keys:

DATABASE_URL=your MONGODB connection string
PORT=your free port.

3. run the following commands:

npm i
npm run build
npm run start:dev

4. For eslint access:

npm run lint
npm run lint:fix
